라이선스 전문과 추가 고지 조건은 각 패키지에 포함된 `LICENSE`, `NOTICE`, `package.json` 파일을 최종 기준으로 확인해야 합니다.

## 아이콘 및 이모지 사용

앱의 분류 아이콘은 `@expo/vector-icons`를 통해 제공되는 Feather Icons를 사용합니다. Feather Icons는 MIT License로 제공되는 오픈소스 아이콘 세트입니다.

앱에서 유니코드 이모지를 텍스트로 표시하는 경우, 이모지 이미지를 별도 에셋으로 복제하거나 배포하지 않고 각 운영체제의 기본 렌더링을 사용합니다.
