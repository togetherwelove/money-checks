# 달력 지출 색상 정책

## 상태

- 정책 결정일: 2026-07-23
- 상태: 확정

## 적용 범위

- 날짜 셀 안에 표시되는 지출 금액의 텍스트 색상에만 적용한다.
- 월간 요약, 선택 날짜 내역, 전체 내역, 차트 등 다른 지출 표시는 기존 지출색을 유지한다.
- 조회 전용 상태의 금액은 기존처럼 비활성 텍스트 색상을 우선 적용한다.

## 선택지와 기본값

- `검은색`: 앱 기본 텍스트 색상을 사용한다.
- `빨간색`: 기존 지출색을 사용한다.
- 신규 사용자와 설정값이 없는 기존 사용자의 기본값은 `검은색`이다.

## 설정 UI

- `앱 설정 > 기능` 섹션에 `달력 지출 색상` 항목을 추가한다.
- 검은색과 빨간색을 즉시 비교할 수 있는 인라인 분할 선택기를 사용한다.
- 각 선택지에는 색상 견본과 텍스트 라벨을 함께 표시한다.
- 선택 결과는 저장 즉시 달력에 반영한다.

## 저장 정책

- 설정은 기기 로컬 저장소에 저장한다.
- 계정이나 다른 기기와 동기화하지 않는다.
- 알 수 없는 저장값은 `검은색`으로 복구한다.
